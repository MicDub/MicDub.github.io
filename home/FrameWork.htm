.Net Framework

In order to use HtmlTweak, you will need to have the Microsoft .Net framework installed on your computer.

Information from Microsoft on how to download the .Net framework can be found on:

http://msdn.microsoft.com/netframework/downloads/howtoget.asp

There are a number of sources from which you can obtain the .Net framework:

  • The .Net Framework is an optional installation included in the Microsoft Windows XP Service Pack 1 (SP1a) stand-alone CD.
  • You can install the framework from Windows Update: http://windowsupdate.microsoft.com/
  • You can download the framework from this Microsoft MSDN download page: http://www.microsoft.com/downloads/details.aspx?FamilyId=D7158DEE-A83F-4E21-B05A-009D06457787&displaylang=en
