The Smooth Scrolling text editor.

SE is the first DOS text editor with FAST Smooth Scrolling technique and interface for Windows. FAST Smooth Scrolling makes moving through text extremely swift and fluid.
Features: copy/cut/paste from/to Windows, macros, virtually unlimited file lengths, multiple documents, multilevel undo, Drag&Drop, symbol table, customizable scrolling and color usage, bookmarks, word wrapping etc.

The latest version is 2.12. This version is a fix for Windows95 and low conventional memory problems and a couple of other bugs.
